## Structure

```
index.html                        # Homepage with race listing
reports/
  epping-forest-50k.html
  country-to-capital.html
  beyond-trails-55k.html
  template.html                   # Blank template for new reports
```

## Adding a new report

1. Copy `reports/template.html` and rename it to your race
2. Fill in everything marked `CHANGE`
3. Add a race card to `index.html`
4. Push — the site updates automatically
